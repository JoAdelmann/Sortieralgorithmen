Anleitung des Versuchslabors

Welcher Button macht was?

  • Klicke auf "Würfeln" um ein zufälliges Array zu erzeugen.
  • Wähle im Menü "Sortieralgorithmen" einen Algorithmus aus.
  • Wenn der Algorithmus automatisch sortieren soll, klicke auf "Animieren". ACHTUNG: Es muss zunächst ein Array erzeugt worden sein!
  • Falls Du den Algorithmus selbst ausprobieren möchtest, klicke zunächst auf "Einzelschritt".
  • Um dann jeweils einen Schritt weiter zu kommen klicke auf "Weiter".
  • Der "Reset"-Button lädt das ursprüngliche unsortierte Array.
  • Der "Clear"-Button bereinigt die Fläche.
